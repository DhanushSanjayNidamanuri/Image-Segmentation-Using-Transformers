# Image Segmentation using Transformers

This repository contains our course project for **CS736: Medical Image Computing**, focused on image segmentation using Transformer-based architectures.

We worked on two different datasets:

- **BTCV** — abdominal organ segmentation  
- **DRIVE** — retinal blood vessel segmentation  

Since the requirements and characteristics of the datasets differ, each uses a slightly different model architecture and training pipeline.

---

## Repository Structure

```text
.
├── btcv/      # BTCV dataset implementation and instructions
├── drive/     # DRIVE dataset implementation and instructions
├── presentation.pdf
└── papers/
```

- `btcv/` contains the code, training pipeline, and setup instructions for the BTCV dataset.
- `drive/` contains the code, training pipeline, and setup instructions for the DRIVE dataset.

Each folder has its own `README.md` explaining:
- dataset preparation,
- environment setup,
- training,
- evaluation, and
- reproduction of results.
